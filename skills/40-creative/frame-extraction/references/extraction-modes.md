# Extraction Modes

Use these modes as routing choices, not as rigid product features.

## `uniform-sample`

Use when:

- the user needs a fast browseable overview
- no stronger intent is present

Typical output:

- frames every N seconds
- one contact sheet

Weakness:

- often captures transitions, blur, or empty frames

## `scene-change`

Use when:

- the user wants shot structure
- pacing or edit rhythm matters
- the video has many cuts

Typical output:

- one or more frames per scene boundary
- timestamp list

Best for:

- shot review
- structure mapping
- benchmark style scans

## `face-priority`

Use when:

- the user wants creator appearance
- persona or vibe matters
- wardrobe, camera distance, lighting, and environment are important

Typical output:

- clear creator-visible frames
- contact sheet
- optional vibe summary

Avoid overclaiming identity or demographics.

## `object-priority`

Use when:

- product presence matters
- the user wants to study reveal, handling, or close-ups

Typical output:

- product-visible frames
- detail frames
- handling/use frames

## `text-ui-priority`

Use when:

- UI readability matters
- the user wants software demos, subtitles, pricing, overlays, or on-screen claims

Typical output:

- readable interface frames
- text-heavy moments
- key-step screenshots

## `hook-first`

Use when:

- the user is studying the opening
- retention or first impression matters

Typical scope:

- first 3-5 seconds

Typical output:

- 3-8 opening frames
- contact strip

## `cta-last`

Use when:

- the user wants ending CTA, pricing, link prompts, claims, or legal/compliance proof

Typical scope:

- last 3-8 seconds

## `before-after-pair`

Use when:

- the video contains a visible transformation
- the user needs comparison frames

Typical output:

- a before frame
- an after frame
- optional side-by-side image

## `style-board`

Use when:

- the user wants mood, lighting, composition, props, or visual-language reference

Typical output:

- a curated multi-frame board
- notes on color, framing, lighting, and texture
