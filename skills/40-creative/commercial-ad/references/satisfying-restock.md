# Satisfying Restock

Template reference: `reboxing`.

## Use When

Use when repetition, abundance, organization, storage fit, color sequence, packaging, or tactile completion is the product appeal.

## Grammar

- Open on an empty or partially empty destination.
- Repeat one placement action with clear spatial progress.
- Insert one slower detail beat for texture, condensation, variety, or package window.
- Show the completed arrangement before closing the drawer, door, lid, or container.
- End on physical closure and visual order.

## Camera, Edit, And Sound

- Use centered, symmetrical, mostly static framing.
- Cut or speed-ramp on repeated placements without changing the spatial rule.
- Synchronize each placement, slide, click, and close to sound.

## Proof And Continuity

- Demonstrate exact fit and abundance visually; do not infer performance claims.
- Lock package size, count, color order, destination geometry, hand direction, and completed layout.

## Avoid

- Impossible package counts or changing storage dimensions.
- Repetition without visible progress.
- Ending before the complete arrangement is shown.

## Generator Risks

Object counts, grid alignment, repeated package geometry, labels, hand continuity, collision-free placement, and exact storage fit.
